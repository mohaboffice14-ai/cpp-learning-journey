# Switch Case Practice

This folder contains C++ programs created while learning switch-case statements.

## Programs

1. Traffic Light Controller
2. Simple Calculator
3. Banking System
4. University Portal
5. Smart Home Control System

## Concepts Practiced

* Switch Case Statements
* Menu-Driven Programs
* Nested Switch Cases
* User Input Handling
* Basic Program Design

## Learning Outcome

Learned how to replace multiple conditional checks with switch-case statements and build interactive menu-based applications.
