# Nested If-Else Practice

This folder contains C++ programs developed while learning Nested If-Else statements.

## Programs

### 1. Admission Eligibility System

Determines admission status based on academic marks and interview performance.

### 2. Employee Bonus Calculator

Calculates employee bonus percentages using salary and experience criteria.

### 3. Traffic Fine System

Calculates traffic fines based on speed limits and license status.

### 4. Gaming Rank System

Assigns player ranks based on level, wins, and losses.

### 5. Scholarship Eligibility System

Determines scholarship eligibility using academic performance, attendance, age, and sports achievements.

## Concepts Practiced

* If-Else Statements
* Nested If-Else Statements
* Multi-Level Decision Making
* Real-World Problem Solving

## Learning Outcome

Learned how to create complex decision-making systems using nested conditional statements.
