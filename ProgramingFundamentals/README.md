# Programming Fundamentals

This section contains programs developed while learning the fundamental concepts of C++ programming.

The programs in this directory focus on understanding program flow, decision-making, user input handling, and problem-solving techniques.

## Topics

* Nested If-Else
* Switch Case
* Loops
* Functions
* Arrays
* Strings
* Pointers

Each topic is organized into its own folder and contains practice programs that demonstrate the concepts being studied.
